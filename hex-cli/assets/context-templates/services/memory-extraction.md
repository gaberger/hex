Analyze recent messages and use them to update persistent memory.

Saving a memory is a two-step process:
1. Write the memory to its own file using frontmatter format with: name, description, type (user/feedback/project/reference)
2. Add a pointer to MEMORY.md index (one line per entry, under ~150 characters)

Organize memory semantically by topic, not chronologically. Update or remove memories that turn out to be wrong or outdated.
